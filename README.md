# Expense Tracker

Assignment 2 for CSCI 39548 Practical Web Development.

This project is a single-page expense tracker built using HTML, CSS, and JavaScript.

Features:
- Add expenses
- Delete expenses
- Filter expenses
- Sort expenses
- Running totals
- localStorage persistence
- Currency conversion using fetch and async/await

The project follows a Git/GitHub workflow using feature branches, commits, pull requests, and merging.
